Setup Wizard - Installation Guide ✕

🛠️ インストール手順 (デベロッパーモード)

  1. GitHubから最新のソースコードをダウンロードします。
    ソースをダウンロード (GitHub)
  2. Chromeを開き、アドレスバーに chrome://extensions と入力します。
  3. 右上の「デベロッパーモード」スイッチをONにします。
  4. 「パッケージ化されていない拡張機能を読み込む」ボタンをクリックします。
  5. ダウンロードして展開した Zero-G-Ghost フォルダを選択します。
  6. 新しいタブを開き、Zero-G Ghostの世界を楽しんでください！

🛠️ Installation Steps (Developer Mode)

  1. Download the latest source code from GitHub.
    Download Source (GitHub)
  2. Open Chrome and enter chrome://extensions in the address bar.
  3. Switch on the "Developer mode" toggle in the top right.
  4. Click the "Load unpacked" button.
  5. Select the extracted Zero-G-Ghost folder.
  6. Open a new tab and enjoy the world of Zero-G Ghost!
Developer Console - Technical Insight ✕

🧠 テクニカル・ハイライト

Visibility-Auto: ブラウザネイティブの C++ 層で描画を制御し、JS 実行を最小化。

3D Scan Analysis: elementsFromPoint を活用し、自分より上に重なっている不透明な要素（遮蔽物）をミリ秒単位で特定、その下の描画を完全に停止します。

🧠 Technical Highlights

Visibility-Auto: Native C++ layer controlling rendering, minimizing JS overhead.

3D Scan Analysis: Leverages elementsFromPoint to identify opaque blockers layered above elements in milliseconds, completely halting rendering underneath.
